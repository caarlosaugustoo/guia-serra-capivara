# Guia Serra da Capivara

Projeto acadêmico desenvolvido para a disciplina de **Front-End 1**, com o objetivo de criar um guia turístico sobre a região da **Serra da Capivara**, localizada no Piauí.

O site apresenta informações sobre pontos turísticos, museus, trilhas, paisagens naturais e locais importantes da região, utilizando apenas **HTML** e **CSS**.

## Preview do projeto

O site possui uma página inicial em formato de landing page, com seções visuais, botões de navegação e páginas individuais para cada ponto turístico.

## Páginas do site

O projeto conta com as seguintes páginas:

- `index.html` — Página inicial do guia
- `pontos-turisticos.html` — Lista dos pontos turísticos
- `contato.html` — Página de contato com formulário
- `parque-nacional.html` — Página sobre o Parque Nacional Serra da Capivara
- `pedra-furada.html` — Página sobre a Pedra Furada
- `museu-homem-americano.html` — Página sobre o Museu do Homem Americano
- `museu-natureza.html` — Página sobre o Museu da Natureza
- `baixao-andorinhas.html` — Página sobre o Baixão das Andorinhas
- `trilhas-mirantes.html` — Página sobre trilhas e mirantes

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsividade com Media Queries
- Fonte Montserrat

## Funcionalidades

- Navegação entre páginas
- Página inicial com seções em largura completa
- Cards de pontos turísticos clicáveis
- Páginas individuais com informações detalhadas
- Galerias de imagens
- Formulário de contato
- Layout responsivo para telas menores

## Conceitos aplicados

Durante o desenvolvimento do projeto foram aplicados conceitos básicos e intermediários de HTML e CSS, como:

- Estruturação semântica com `header`, `main`, `section`, `article`, `aside` e `footer`
- Organização de layout com Flexbox
- Organização de cards e galerias com CSS Grid
- Uso de imagens com `object-fit`
- Estilização de botões, cards e formulários
- Responsividade com `@media`
- Separação entre estrutura HTML e estilos CSS
